# PvPToggle
Kaai's PvPToggle-Plugin

Permissions:
- pvptoggle.pvp.toggle
  - allows to toggle your pvp state
- pvptoggle.pvp.other
  - allows to toggle others pvp state
- pvptoggle.pvp.state
  - allows to show yours pvp state
- pvptoggle.pvp.list
  - allows show a global pvplist
